# Test List

Простой тестовый проект, собранный с использованием Vite.

## Установка

Для начала работы с проектом, необходимо установить зависимости с помощью команды:

```bash
npm install
```

Запуск
После успешной установки зависимостей, запустите проект командой:

```bash
npm run start
```

Откройте ваш браузер и перейдите по адресу http://localhost:5173 для просмотра приложения в действии.
